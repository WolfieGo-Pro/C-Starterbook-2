# C++ Starterbook-2

Continuation from C-Starterbook.

NOTE:

Each of the source files will need a 'pch.h' header file where all the important header files can be included.

Example:

pch.h

#define PCH_H
#endif #PCH_H

// files needed
#include <iostream>
#include <string>
#include <iomanip>
#include <limits>

#endif //PCH_H

ALTERNATIVELY,

The 'files needed' can be added directly in each source file.
